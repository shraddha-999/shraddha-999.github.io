RateGuard

Distributed Rate Limiter + SaaS Dashboard

A production-ready distributed rate limiter with a full React 19 monitoring dashboard. Three algorithms — Fixed Window, Sliding Window Log, and Token Bucket — all backed by atomic Lua scripts in Redis (EVALSHA) for race-condition-free operation at any concurrency level.

  • Atomic Lua scripts via EVALSHA — no WATCH/MULTI/EXEC overhead, correct under concurrent requests
  • In-memory Lua emulator for tests; 49 tests run at full speed without a Redis process
  • Strategy-pattern algorithm factory + hot-reload config via PUT /config/{id} with no restart
  • React 19 dashboard: real-time charts, algorithm visualizer, request simulator, Prometheus metrics, CSV log export
  • Free-tier deploy: Vercel (frontend) + Render (FastAPI) + Upstash Redis
